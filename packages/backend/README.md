# @cap2ui5/backend

CAP backend module for cap2UI5 apps.

> **Source lives in [`cap2UI5/dev`](https://github.com/cap2UI5/dev)** under
> `packages/backend/`. The standalone repo `cap2UI5/backend` is a read-only
> mirror — open issues and PRs against `cap2UI5/dev`.

## Install

```bash
npm install @cap2ui5/backend
```

## Migration note

The full CAP project (db/, srv/, mta.yaml, …) currently lives in
[`cap2UI5/backend`](https://github.com/cap2UI5/backend) on `main` and will be
migrated here under `packages/backend/`.
