# @cap2ui5/frontend

UI5 frontend library for cap2UI5 apps.

> **Source lives in [`cap2UI5/dev`](https://github.com/cap2UI5/dev)** under
> `packages/frontend/`. The standalone repo `cap2UI5/frontend` is a read-only
> mirror — open issues and PRs against `cap2UI5/dev`.

## Install

```bash
npm install @cap2ui5/frontend
```
